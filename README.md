# Proyecto API para la Venta de Boletos de Cine



## **Contexto del Negocio**
Sistema de gestión para cines que permite la venta de boletos para proyecciones cinematográficas, controlando la disponibilidad de asientos en tiempo real para evitar sobreventas.

---

## **Tecnologias Utilizadas**
- Java 17
- Graddle
